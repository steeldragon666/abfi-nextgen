# ABFI Assurance Pack — Export Specification v1

**Design Philosophy:** Lender-ready · Audit-defensible · Plain-English

---

## 1) Purpose & Positioning

### What This Is

A standardized, evidence-backed assurance summary for a contracted bioenergy feedstock supply.

### What This Is NOT

| Not This | Why |
|----------|-----|
| Investment memo | ABFI provides data, not advice |
| Financial forecast | No projections included |
| Legal advice | Disclaimer required |
| Substitute for due diligence | Users verify independently |

### How Lenders Use It

| Use Case | Context |
|----------|---------|
| Internal credit committee review | Decision support |
| Ongoing covenant / risk monitoring | Portfolio oversight |
| Project finance assurance attachment | Documentation |
| Government / grant reporting | ARENA, CEFC, etc. |
